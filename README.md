# Shelfmark Home Assistant Addon

Home Assistant addon for [Shelfmark](https://github.com/calibrain/shelfmark) - a unified book and audiobook download aggregator.

## About

Shelfmark provides a single interface for searching and downloading books and audiobooks from multiple sources including web archives, torrents, Usenet, and IRC.

## Installation

### Add Repository

1. Go to **Settings** > **Add-ons** > **Add-on Store**
2. Click the three dots menu (top right) > **Repositories**
3. Add this repository URL
4. Click **Close** and refresh the page
5. Find "Shelfmark" in the add-on list and install it

### Configuration

After installation:

1. Configure your timezone in the add-on settings
2. Start the add-on
3. Click "Open Web UI" or access from the sidebar

## Features

- Integrated in Home Assistant sidebar via ingress
- Downloads saved to `/media/shelfmark` (accessible via Media Browser)
- Persistent configuration stored in add-on data
